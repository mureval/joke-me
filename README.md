# JOKE ME
This application can display a joke taken from the PUBLIC API: [JOKE API](https://sv443.net/jokeapi/v2/)

### GETTING STARTED
---
1. Open `terminal` or `git bash`
2. type `NPM INSTALL`
3. type `NPM RUN DEV`
4. Open `http://localhost:3000` in your web browser

### DEPENDENCY
---
* Express
* body-parser
* EJS `Embeded JavaScript`
* nodemon
